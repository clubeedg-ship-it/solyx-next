Solyx Energy — Website

Every page below is a plain, standalone HTML file. This is the working copy of the site where the content gets finished before it goes live.

How to edit: there is no build step. Open any .html file directly in a browser to preview it, and open the same file in an editor to change the text, images and links. Save the file and refresh the browser to see the change.
Working with the assistant: just name the page and what you want changed (e.g. "on besparen.html, change the hero heading and the three bullet points") and the assistant edits the HTML for you.
